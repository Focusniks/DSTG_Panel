SSH ключ для приватных репозиториев
Зачем это нужно?

Если вы используете приватные репозитории на GitHub для ваших ботов, необходимо настроить SSH ключ для доступа.

Этот ключ будет использоваться для всех ботов в панели.

Ошибка "Key is already in use"?

Эта ошибка возникает, если вы пытаетесь добавить один и тот же SSH ключ как deploy key к нескольким репозиториям.

Решение: Добавьте ключ к вашему аккаунту GitHub (Settings → SSH and GPG keys), а не к каждому репозиторию отдельно. Один ключ, добавленный к аккаунту, будет работать для всех ваших репозиториев.
